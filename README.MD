# athena（雅典娜）java微服务架构+react前后端分离 

## 整体项目采用springcloud 分布式框架

    模块一览
|模块名|对应的模块|
|-|-|
|eureka|服务注册中心模块|
|gateway|网关模块|
|common|公共引用模块|
|uc|系统权限模块|


    依赖一览

|jar包名称|备注|
|-|-|
|spring-cloud-starter-eureka|springcloud注册中心|
|spring-cloud-starter-zuul||
|spring-boot-devtools|springboot热部署|
|mybatisplus-spring-boot-starter|mybatis扩展|
|lombok| 减少java代码量|
|druid| 阿里数据库连接池|
